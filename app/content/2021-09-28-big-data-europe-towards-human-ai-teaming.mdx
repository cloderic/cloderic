The [Big Data Conference Europe](https://bigdataconference.eu) is a yearly event focused on the latest advancements and applications in Big Data, High Load, Data Science, Machine Learning, and Artificial Intelligence.

This talk, copresented with [Sagar Kurandwad](https://www.linkedin.com/in/sagar-kurandwad-0936a827/), explores the integration of human insights into AI systems. We discussed the design of hybrid human-AI systems trained from human-AI interactions using human-in-the-loop learning techniques (HILL).

For more details, [you can read more about the talk here](https://bigdataconference.eu/2021/Cloderic-Mars/index.html).
